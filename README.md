# かいわ

ブラウザだけで文字会話をつなぐ、聴覚障害者向けの会話支援アプリです。

## できること

- 同じ URL を開いた 2 台の端末を接続する
- QR または共有コードで接続情報を受け渡す
- 接続後はブラウザ間の P2P 通信で文字会話する
- 相手の入力中テキストを下書きとして表示する

## 使い方

1. 本人側と会話相手側で同じ URL を開く
2. 本人側で `この端末で開始` を押す
3. 会話相手側で `同じ URL で参加` を押す
4. QR または共有コードで接続する
5. 会話画面で文字を送受信する

## 技術構成

- 配信: GitHub Pages
- 通信: WebRTC DataChannel
- QR 表示: qrcodejs
- QR 読み取り: BarcodeDetector / jsQR fallback

## 補足

- 一般利用ではアプリのインストールは不要です
- QR が使いにくい場合でも共有コードのコピーと貼り付けで接続できます
- このリポジトリは公開用の最小構成で、開発用の tests / docs は含めていません
